## 📌 Project Overview

This repository contains a machine learning project designed to predict potential diseases based on a given set of patient symptoms. By utilizing supervised learning techniques, the model analyzes patterns in the symptom data to output a predicted diagnosis. 

## 🗄️ Dataset

The project relies on two primary datasets:
* `Training.csv`: Used to train the machine learning model. It contains various symptoms as features and the corresponding disease as the target label.
* `Testing.csv`: Used to evaluate the model's accuracy and performance on unseen data.
